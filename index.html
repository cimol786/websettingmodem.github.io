💡 Apa Itu Modem WiFi?

Modem adalah perangkat pintar yang menghubungkan rumah Anda ke jaringan internet luas milik Life Media, sekaligus memancarkan gelombang sinyal WiFi ke smartphone, laptop, hingga Smart TV Anda.

Silakan pilih tipe paket modem Anda:

Contoh Unit Standar Modem 30 Mbps
Langkah 01

Hubungkan & Akses IP

Buka browser Google Chrome di HP/Laptop yang sudah terhubung ke WiFi. Ketik alamat IP 192.168.1.1 di kolom pencarian atas. Isi Username: admin dan Password: admin.

Langkah 02

Masuk Menu WLAN

Setelah berhasil masuk ke dashboard internal, arahkan navigasi ke menu Network kemudian pilih submenu WLAN sesuai petunjuk gambar.

Langkah 03

Ganti Nama WiFi (SSID)

Pilih tab SSID Setting. Ubah nama jaringan Anda di kotak teks SSID Name. Setelah diganti, geser ke bawah lalu klik tombol Submit di sudut kanan bawah gambar.

Langkah 04

Ganti Password WiFi

Pilih tab Security. Hapus kata sandi lama pada bagian WPA Passphrase dan ketikkan kata sandi baru minimal 8 karakter. Terakhir, klik kembali tombol Submit.

Langkah 01

Akses Halaman Login Gateway

Buka Chrome, lalu masukkan IP khusus 192.168.101.1. Isi Username: admin dan Password: admin pada halaman login.

Langkah 02

Konfigurasi Dual-Band (Wi-Fi Basic)

Pilih menu Wi-Fi > Basic. Di sini Anda bisa mengubah nama pada bagian Wi-Fi Name dan kunci pengaman pada Wi-Fi Password. Sifat dual-band mengharuskan penamaan frekuensi 2.4G dan 5G dibedakan sedikit. Klik Submit/Sure untuk menyimpan.

🙋 Pertanyaan yang Sering Diajukan (FAQ)

Mengapa halaman IP tidak bisa dibuka? ▼
Pastikan perangkat HP atau Laptop Anda sudah benar-benar terhubung ke sinyal WiFi modem tersebut, bukan menggunakan paket data seluler biasa.
Berapa digit minimal password WiFi? ▼
Minimal password adalah 8 karakter. Kombinasikan huruf, angka, dan simbol unik agar WiFi Anda tidak mudah dibobol.
